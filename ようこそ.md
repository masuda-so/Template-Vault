# Template Vault へようこそ

このリポジトリは **Obsidian 用スターター Vault** です。  
clone してすぐに日々のノート作りを始められるよう、フォルダ構成・テンプレート・Bases ビューを事前に用意しています。

## 最初にやること（3 ステップ）

### ステップ 1: このファイルを読む（今まさにやっています！）

下の「Vault フォルダツアー」を眺めて、どこに何を置くかを把握しましょう。

### ステップ 2: Obsidian の Core Plugins を有効化する

Settings（歯車アイコン）→ **Core plugins** を開き、以下を ON にしてください。

| プラグイン | 役割 |
| --- | --- |
| **Templates** | `Templates/` フォルダのテンプレートをノートに挿入できるようになる |
| **Daily notes** | `Daily/` に日付ノートをワンクリックで作成できるようになる |
| **Bases** | `Bases/` の `.base` ファイルでノートをテーブル・カード表示できるようになる |

Templates プラグインを ON にしたら、Settings → Templates → **Template folder location** に `Templates` と入力して保存してください。

### ステップ 3: `Ideas/` に最初のアイデアを書いてみる

1. 左サイドバーの `Ideas` フォルダを右クリック → **New note**
2. タイトルに思い浮かんだアイデアを書く
3. コマンドパレット（`Ctrl/Cmd + P`）→「Templates: Insert template」→ `idea` を選んでフォームを挿入する

それだけです。あとは自分のペースで使い始めましょう。

## Vault フォルダツアー

| フォルダ | 置くもの |
| --- | --- |
| `Daily/` | 毎日の記録・ログ・タスク書き出し |
| `Weekly/` | 週次計画と振り返り |
| `Monthly/` | 月次計画と振り返り |
| `Projects/` | 実行中プロジェクトのノート |
| `Ideas/` | まだ形が固まっていないアイデアのメモ |
| `References/` | 長期的に再利用する知識・まとめノート |
| `Clippings/` | Web 記事の抜粋・一次ソース保存 |
| `Templates/` | 各フォルダ向けの挿入用テンプレート |
| `Bases/` | ノートをテーブル・カードで見る `.base` ファイル |
| `Sandbox/` | Obsidian の書式・機能を試す練習スペース |
| `Canvases/` | 視覚的なキャンバス（マインドマップなど） |
| `Meta/` | Vault 運用ルールや改善メモ |
