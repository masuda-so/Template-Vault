# Clippings

Web や資料から取得した抜粋、引用、一次ソースを保存するフォルダです。

## このフォルダでできること

- 記事やページの重要箇所を保存
- 後で要約・再整理するための素材を蓄積
- 参照ノート化する前段階の情報を一時保管

## このフォルダが役立つ理由

- 情報収集と知識整理を分離できる
- 収集時点では雑でも、後で [../References](../References) に昇格しやすい
- 出典ベースでノートを作れるため、再検証しやすい

## はじめ方

1. Web クリップや手動メモをこのフォルダに保存
2. ノート名に日付や出典を含める
3. 週次レビュー時に要点を [../References](../References) へ抽出

## 利用例

保存例:

```md
---
type: clipping
source: https://example.com/article
captured: 2026-04-01
---

# Obsidian 運用記事メモ

- 引用1
- 引用2
```
