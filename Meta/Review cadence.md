# Review cadence

Checklists for daily, weekly, and monthly reviews in this vault.

## Daily review

Run at the start or end of each day.

- [ ] Open today's daily note in `Daily/` (create one if it does not exist).
- [ ] Write down the top one to three things to focus on today.
- [ ] Scan `Ideas/` for any quick captures from the previous day that need context added.
- [ ] Check open tasks from yesterday's daily note and carry forward anything unfinished.
- [ ] Record notable events, decisions, or observations before they fade.

> [!tip] Keep it short
> The daily review should take five minutes or less. Its purpose is focus and capture, not planning.

## Weekly review

Run once per week, ideally on a consistent day such as Friday.

- [ ] Create a new weekly note in `Weekly/` using the `YYYY-Www` format.
- [ ] Review all open projects in `Projects/` and update their status.
- [ ] Scan `Ideas/` and decide the fate of each note: promote, keep, or discard.
- [ ] Identify any idea ready to become a project and move it to `Projects/`.
- [ ] Review last week's weekly note for unresolved follow-ups.
- [ ] Identify any project that should be archived (see [[Archive policy]]).
- [ ] Plan the top priorities for the coming week.

> [!tip] Promote only when ready
> An idea should survive at least two weekly reviews before becoming a project. Premature promotion creates clutter in `Projects/`.

## Monthly review

Run once per month, ideally on the last working day of the month.

- [ ] Create a new monthly note in `Monthly/` using the `YYYY-MM` format.
- [ ] Review all active projects in `Projects/` for progress and relevance.
- [ ] Identify projects that are complete or should be archived.
- [ ] Promote durable knowledge from `Projects/` or `Clippings/` to `References/`.
- [ ] Review `References/` for notes that have become outdated and should be archived.
- [ ] Assess the overall vault structure: are folders still appropriate for current usage?
- [ ] Check [[Folder conventions]] and [[Archive policy]] for any needed updates.
- [ ] Set top priorities for the coming month.

> [!note] Vault health check
> The monthly review is the right time to update [[Folder conventions]] or [[Archive policy]] if your workflow has drifted from the documented conventions.
