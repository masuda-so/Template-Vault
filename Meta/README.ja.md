# Meta

[English version](README.md)

Vault の運用方針、命名規則、改善履歴など「仕組みそのもの」を記録するフォルダです。

## このフォルダでできること

- 命名規則の定義
- フォルダ運用ルール
- レビュー手順
- プラグイン採用方針
- 構造変更の履歴メモ

## このフォルダが役立つ理由

- プロジェクト内容と運用ルールを分離できる
- 将来の自分や共同編集者が意思決定理由を追える
- Vault 改善を再現可能な形で進められる

## はじめ方

構造や運用に関する意思決定をしたら、その理由と運用方法をこのフォルダに記録します。

## 利用例

```md
# フォルダ運用規約

## Projects
- 実行中の作業のみ
- 原則 1 プロジェクト 1 ノート

## References
- 長期参照する知識ノート

## レビュー頻度
- 毎週金曜に週次レビュー
- 月末に月次レビュー

> [!note] 運用ガバナンス
> フォルダ定義を変更したらこの規約を更新し、[[README#利用例]] へ相互リンクする。

## チェックリスト
- [ ] 命名規則が現行フォルダ構成と一致しているか確認
- [ ] [[Sandbox/README]] の導線リンクを更新
```
