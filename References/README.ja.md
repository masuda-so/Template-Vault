# References

長期的に再利用する知識ノートを蓄積するフォルダです。

## このフォルダでできること

- エバーグリーンノートの作成
- 情報源の要約と再整理
- 技術メモ・用語集の管理
- 書籍・記事・プロジェクトから得た知見の保存

## このフォルダが役立つ理由

- 実行中作業から独立した知識資産を育てられる
- 後から再利用しやすい形で情報を残せる
- Ideas / Projects の背景文脈を安定供給できる

## はじめ方

将来の自分が再利用できる密度で、問いに答える形式のノートを作成します。

```md
---
type: reference
topic: obsidian
---

# 週次レビューのチェックリスト

## 目的
Ideas・Projects・計画ノートを毎週同期する。

## 手順
- 進行中プロジェクトを確認
- 停滞アイデアを昇格または保留
- 翌週の障害要因を記録
```
