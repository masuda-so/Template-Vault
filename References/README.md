# References

Keep durable notes here: material you expect to revisit, connect, and reuse over time.

## What This Folder Does

Use this folder for reference notes such as:

- evergreen concepts
- source summaries
- technical how-tos
- glossaries
- distilled lessons from books, articles, or projects

Unlike [Projects](../Projects/README.md), reference notes should remain useful even when no active task depends on them.

## Why This Folder Is Useful

- Separates long-term knowledge from active work.
- Makes it easier to build a reusable knowledge base instead of a pile of temporary notes.
- Gives ideas and projects a stable place to link to supporting context.

## How To Get Started

Create notes that answer a question clearly enough for your future self to reuse.

Example reference note:

```md
---
type: reference
topic: obsidian
---

# Weekly review checklist

## Purpose
Keep project, idea, and planning notes aligned once per week.

## Checklist
- Review active projects
- Promote or archive stale ideas
- Note blockers for next week
```
