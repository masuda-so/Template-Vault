# Weekly

週次の計画と振り返りを行い、日次メモと月次方針をつなぐフォルダです。

## このフォルダでできること

- 進行中プロジェクトの棚卸し
- アイデアの昇格判断
- 次週の優先順位とアクション整理
- 課題・成果・フォローアップの記録

## このフォルダが役立つ理由

- 継続可能なレビューリズムを作れる
- アイデアやプロジェクトの停滞を防げる
- 月次計画に必要な中間データを安定供給できる

## はじめ方

週ごとに 1 ノート作成します（例: `2026-W14`）。

```md
---
type: weekly
week: 2026-W14
reviewed: 2026-03-31
---

# 第14週レビュー

## 進行中プロジェクト
- Vault README 整備

## 昇格候補アイデア
- 週次レビュー質問テンプレート

## ブロッカー
- フォルダ規約が曖昧

## 来週の重点
- ドキュメント更新を完了
```
