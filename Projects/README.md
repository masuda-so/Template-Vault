# Projects

[日本語版はこちら](README.ja.md)

Track active work that needs multiple steps, decisions, or follow-up.

## What This Folder Does

Use this folder for project notes such as:

- active initiatives
- deliverables in progress
- personal experiments with a clear outcome
- work that needs next actions, deadlines, or related resources

If a note is still vague, start it in [Ideas](../Ideas/README.md). If it becomes long-term reference material, extract the durable parts into [References](../References/README.md).

## Why This Folder Is Useful

- Keeps action-oriented work separate from evergreen knowledge.
- Makes reviews easier because all active commitments live in one place.
- Pairs well with the project's Bases definitions for project-oriented views.

## How To Get Started

Create one note per active project and keep the current status near the top.

## Usage Example

```md
---

status: active
```
